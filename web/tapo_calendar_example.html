TaPo Calendar Example

This is a simple example of the calendar component, based on polymer.dart. It allows to display events on a day view. Only one event is allowed for every time (no overlaps yet). It basically just fullfills your needs at WorkTrail, a easy time tracking application (allow manipulation of past work entries).

To begin, simply click and drag!

View the code at github: https://github.com/hpoul/tapo-calendar/
